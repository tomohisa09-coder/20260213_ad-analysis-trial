## 分析レポートの構成（必ずこの順序で出力すること）

### 1. エグゼクティブサマリー
- 総合ランキング（テーブル形式：順位、クリエイティブ名、総合評価、最大の強み、最大の課題）
- 最重要発見（3つ、箇条書き、定量根拠付き）

### 2. パフォーマンス分析
- 各クリエイティブのKPI考察（CTR、CPC、CPA、ROASの比較と解釈）
- 配信継続/停止の要因分析

### 3. 視聴維持率分析
- 各クリエイティブの視聴ドロップオフの特徴
- 最大離脱ポイントとその原因仮説

### 4. クリエイティブ構造×パフォーマンス クロス分析
- フック手法とCTRの関係
- 動画尺とコスト効率の関係
- 権威性・社会的証明等のクリエイティブ要素がKPIに与える影響
- 勝ちクリエイティブの構造的要因分析

### 5. 敗因分析
- パフォーマンスが低いクリエイティブの構造的な問題点
- クリエイティブ構造との因果関係

### 6. 次回クリエイティブ制作への推奨事項

**このセクションは必ず以下の3つのサブセクションすべてを出力すること。省略不可。**

#### 6-1. 勝ちパターンの構造テンプレート
- 今回の分析で判明した「勝ちクリエイティブ」の構造を再現可能なテンプレートとして提示
- 具体的な秒数配分（フック○秒→ボディ○秒→CTA○秒）
- 各パートで使うべき演出・訴求要素を具体的に記載
- テーブル形式で「パート / 秒数 / 内容 / 演出ポイント」を整理

#### 6-2. 具体的な改善アクション（優先度順）
- 優先度High/Medium/Lowに分けて記載
- 各アクションに「期待効果」と「実施難易度」を付与
- テーブル形式：優先度 / アクション / 対象クリエイティブ / 期待効果 / 実施難易度

#### 6-3. 検証すべき仮説
- 今回のデータから導かれる仮説を3〜5個提示
- テーブル形式：仮説 / 検証方法（A/Bテスト設計等） / 期待効果 / 必要素材

## 出力ルール
- Markdown形式で出力
- 数値は必ず定量的根拠として記載（「高い」ではなく「2.20%」）
- テーブルを積極的に使用
- 各セクションに「なぜそうなるのか」の因果関係を必ず記述
- 推奨事項は実行可能で具体的なものにすること
- 定性分析（専門家レビュー）がある場合は、その知見もクロス分析に活用すること
